# Markdoc

Markdoc은 매우 가벼운 Markdown 기반 위키 시스템입니다. 원본 소스가 더 이상 유지 보수되지 않으며
필요한 부분을 수정해서 사용하기 위해 포크 하였습니다.

## 수정된 부분

- 기존에는 xhtml 문서 형태로 제공되었던 부분을 모두 html 문서로 수정

## Installation

    $ python setup.py install
